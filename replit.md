# Crits n Shits - Board Game Club Website

## Overview
A static website for "Crits n Shits" board game club. The site displays information about the club, available games (board games, RPGs, wargames), club masters, gallery, and rules.

## Project Structure
- `index.html` - Main page
- `boardgames-full.html` - Full board games catalog
- `gallery-full.html` - Full gallery page
- `css/style.css` - Main stylesheet
- `js/data-loader.js` - JavaScript for loading game data from JSON
- `js/scroll-spy.js` - Navigation scroll behavior
- `data/` - JSON data files for games, masters, gallery
- `images/` - Image assets (logo, game images, gallery)

## Tech Stack
- Static HTML/CSS/JavaScript
- No build process required
- Data loaded dynamically from JSON files

## Running Locally
Uses Python's built-in HTTP server via `server.py` on port 5000.

## Deployment
Configured as a static site deployment serving from the root directory.
